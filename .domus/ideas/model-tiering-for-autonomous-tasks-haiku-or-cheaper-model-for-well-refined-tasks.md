# Idea: Model tiering for autonomous tasks — haiku or cheaper model for well-refined tasks

**Captured:** 2026-03-17
**Status:** raw

---

## The Idea

If a task is refined to autonomous with enough precision and narrow scope, a smaller/cheaper model (Haiku or equivalent) might be able to execute it reliably. Well-refined tasks with explicit steps, clear acceptance criteria, and tight scope reduce the reasoning burden enough that a capable smaller model could handle it. This could be a meaningful cost saving at scale.
