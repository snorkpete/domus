# Idea: Context alignment checker — keep session context reflecting current reality

**Captured:** 2026-03-17
**Status:** raw

---

## The Idea

As the project evolves, context files (agent-instructions, memory, decisions) can drift out of sync with actual code and design. Need a mechanism (possibly Doctor persona or dedicated persona) to periodically audit context files, identify stale or contradictory content, and either update or archive it. Old context isn't deleted — archived with progressive disclosure so historical info is accessible but not noise in current sessions.
