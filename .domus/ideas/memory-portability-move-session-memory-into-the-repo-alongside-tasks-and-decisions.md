**Captured:** 2026-03-17
**Status:** raw

## The Idea

Currently memory lives in ~/.claude/projects/ and doesn't travel with the repo. For a new machine or collaborator to get up to speed, the 'meat' of context should live in .domus/ and be version-controlled like tasks, ideas, and decisions. Explore whether this overlaps with specs or is a distinct concept. The ~/.claude memory mechanism could remain as an auto-loading convenience that points into the repo.
