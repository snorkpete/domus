## What This Task Is

When refining an autonomous task, the output should include not just acceptance criteria but also a defined step list — the checkpoints the worker will log via domus task log. Steps make execution logs useful for resumability and give the human visibility into progress granularity. Needs a convention for how steps are expressed in the task markdown and how workers reference them.
